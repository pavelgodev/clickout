# Most Viewed Articles

A WordPress plugin that tracks and displays the most viewed articles.

## Description

Most Viewed Articles is a WordPress plugin that automatically tracks page views and allows you to
display most popular content using widget Most Viewed Articles.

## Installation

1. Upload the `most-viewed-articles` folder to the `/wp-content/plugins/` directory
2. Run `composer install` in the plugin directory
3. Run `npm install` in the plugin directory
4. Run `npm run build` in the plugin directory
5. Run `npm run start` in the plugin directory for development
6. Activate the plugin through the 'Plugins' menu in WordPress
7. Activate the plugin through the 'Plugins' menu in WordPress
8. Add widget Most Viewed Articles to any widget area

## Usage

### Widget

Add the Most Viewed Articles widget to any widget area through Widgets page.
