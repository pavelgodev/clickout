# Strategy WordPress Theme

A modern WordPress theme.

## Build Process
Run `npm install` to install the dependencies.
Run `npm run start` to start the development.
Run `npm run build` to build the theme.

## Installation

1. Download the theme files
2. Navigate to WordPress Dashboard > Appearance > Themes
3. Click "Add New" then "Upload Theme"
4. Choose the theme zip file and click "Install Now"
5. After installation, click "Activate"

## Requirements

- WordPress 6.8 or higher
- PHP 8.0 or higher
- MySQL 5.6 or higher
- ACF should be installed and activated
